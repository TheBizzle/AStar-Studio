AStar-Studio
===============

## What is it?

A web application that wraps around [my Golang A* implementation](https://github.com/TheBizzle/AStar-Golang).

Overkill?  Yeah, probably.

Pointless?  Yeah, probably.

But also educational!

## Why is it?

I wanted to actually give Go's concurrency constructs a test drive, so I contrived something that could build upon my other work with Go. 🤷‍♂️
